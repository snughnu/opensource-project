# 개요

- 언어: python
- 기여자: 오승민

# 기능 설명

## 기본 기능

블록, 공, 패들이 존재하며 다음과 같은 기능 등이 있습니다.
- 공과 블록이 부딪치면 블록이 제거되는 기능
- 공과 벽이 부딪치는 기능

## 아이템 기능

블록이 깨지면 20%의 확률로 빨간색이나 파란색의 아이템이 떨어지는 기능이 있습니다.
- 빨간색은 paddle에서 공하나가 추가로 발사
- 파란색은 기능 미구현
